Is Prime
Limits 1s, 512 MB

Given an integer N N N, determine if it is a prime a number.

A number is called prime if it is only divisible by itself.

Input

The input will contain one integer N N N ( 0 < N < 1000 0 < N < 1000 0<N<1000).

Output

Print Yes \texttt{Yes} Yes if the integer is prime, otherwise No \texttt{No} No.

Sample

Input Output 
7
 
Yes
